# API Vendo — informacje dla agentow

## Dokumentacja API

Pelny opis endpointow API Vendo znajduje sie w pliku `swagger.json` w folderze glownym projektu. Zawiera definicje wszystkich dostepnych endpointow, modeli wejsciowych/wyjsciowych i ich pol. Przed implementacja nowego endpointu lub modyfikacja istniejacego — sprawdz swagger.json, zeby uzyc prawidlowych nazw pol i sciezek.
